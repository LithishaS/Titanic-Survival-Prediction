# 🚢 Titanic Survival Prediction

A Machine Learning project that predicts whether a passenger survived the Titanic disaster using Logistic Regression and the Titanic dataset.


## 📌 Project Overview

This project analyzes passenger information from the Titanic dataset and builds a machine learning model to predict passenger survival.

The project includes data preprocessing, exploratory data analysis, feature engineering, model training, evaluation, and prediction.


## ✨ Features

- Data preprocessing
- Missing value handling
- Exploratory Data Analysis (EDA)
- Data visualization
- Feature encoding
- Logistic Regression model
- Survival prediction


## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## 📊 Dataset

The project uses the Kaggle Titanic dataset.

Files included:

- train.csv
- test.csv

Dataset features include:

- Passenger ID
- Passenger Class
- Name
- Sex
- Age
- Fare
- Embarked
- Family Information
- Survival Status (Training Data)


## ⚙️ Workflow

1. Load the Titanic dataset
2. Handle missing values
3. Perform exploratory data analysis
4. Encode categorical features
5. Split features and target
6. Train a Logistic Regression model
7. Evaluate model performance
8. Predict passenger survival


## 📈 Machine Learning Algorithm

- Logistic Regression
